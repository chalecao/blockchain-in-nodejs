手把手教你用JavaScript编写区块链，理解区块链技术的同时，基于NodeJS编写自己的区块链，实现区块创建与挖掘，数据同步等。

第一章 区块链与挖矿概念

2、挖矿与区块链分叉

1、确定区块链与区块数据结构
2、交易记构建流程和代码实现

第三章 POW（工作量证明）实现

2、封装区块链REST API
